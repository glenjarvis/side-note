SideNote

SideNote is a simple JavaScript application that allows you to attach notes to specific parts of a webpage — ideal for use with online books, long-form articles, and academic web content. It provides a clean, accessible way to display annotations as hoverable or clickable popups alongside your text.

Key Features

  • Attach notes to sentences, phrases, or words with ease.
  • Hover- or click-based note display (mobile-friendly).
  • Minimal HTML and CSS required; customizable and lightweight.
  • Supports semantic annotation with accessibility Check spelling in mind.

How to Use SideNote

1. Include SideNote in Your Page

Include the SideNote JavaScript and CSS (if provided) in your HTML file. If SideNote is modular, you may load it via a CDN or local script: 

<script src="sidenote.js" defer></script>
<link rel="stylesheet" href="sidenote.css">

2. Markup Your HTML

Wrap the target text with a span and insert a note using a nested span:

<span class="note-wrapper"> Important term <span class="note-popup"> This is a note about the term. </span> </span>

3. Style Your Notes

You can use basic CSS to control how notes appear. A common setup is showing them on hover or click: 

.note-wrapper {
  position: relative;
  cursor: pointer;
  color: #0070f3;
  text-decoration: dotted underline;
}

.note-popup {
  display: none;
  position: absolute;
  top: 1.5em;
  left: 0;
  background: #fff;
  border: 1px solid #ccc;
  padding: 0.5em;
  border-radius: 5px;
  box-shadow: 0 2px 8px rgba(0,0,0,0.1);
  z-index: 100;
  max-width: 250px;
}

.note-wrapper:hover .note-popup,
.note-wrapper:focus-within .note-popup {
  display: block;
}

Mobile Support

To support touch devices, consider using :focus-within or click-to-toggle behavior with a small JavaScript helper. You can also adjust the position or use a modal-style display for smaller screens.

Advanced Use

SideNote can be extended to auto-inject notes from JSON, markdown files, or CMS sources. With JavaScript, you can dynamically associate text ranges with note metadata, allowing for collaborative annotations or comment layers.
